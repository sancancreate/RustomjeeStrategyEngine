**Rustomjee Sales Strategy Prop-Description Engine**

Features:
- Upload Excel/CSV files
- Automatically detect Property Description column
- Extract:
  - Carpet Area
  - Balcony Area
  - Utility Area
  - Total Area
  - Project Name
  - Building Name
  - Unit Number
  - Floor
  - Parking Information
  - Survey Number
  - RERA Number

Built using:
- Python
- Pandas
- Streamlit
- OpenPyXL
